# wx weeny

An app to observe, learn and stay current with the weather in your location

## Getting Started

1. Install the dependencies:

```
yarn
```

2. Run the server:

```
cd server && yarn start
```

3. Run the app (in a separate terminal window):

```
yarn start
```

4. Publish to Vercel

```
expo build:web && cd web-build && now --prod
```

y -> enter -> y -> wx-weeny
